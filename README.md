# Multiplication problems

Usage:

```bash
       USAGE: ./generate_times_tables.py [flags]
flags:

./generate_times_tables.py:
  --count: Number of questions to generate.
    (default: '20')
    (an integer)
  --high: High value to use.
    (default: '9')
    (an integer)
  --low: High value to use.
    (default: '2')
    (an integer)
  --output: File to output to.
    (default: 'problems')
```
